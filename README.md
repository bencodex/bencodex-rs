[Rust]: https://rust-lang.org/
[Bencodex]: https://bencodex.org/

## Building and Testing

You can build and test the code with [cargo] command.

If you want to build:

```
cargo build
```

If you want to test:

```
cargo test
```

[cargo]: https://github.com/rust-lang/cargo/
